# Musibench — Community

This is the official community repository for [Musibench](https://musibench.com), a browser-based music practice workstation.

> **This repository does not contain the application source code.**
> It exists solely as a space for the community to engage with the project.

## What this is for

- **[Bug reports](https://github.com/marcelnubi/musibench/issues/new?template=bug_report.md)** — something not working as expected
- **[Feature requests](https://github.com/marcelnubi/musibench/issues/new?template=feature_request.md)** — ideas for new tools or improvements
- **[Discussions](https://github.com/marcelnubi/musibench/discussions)** — general questions, practice tips, feedback

## What this is NOT for

- Pull requests with code changes (the source code is not hosted here)
- Support requests unrelated to Musibench

## Getting started

1. Search [existing issues](https://github.com/marcelnubi/musibench/issues) before opening a new one
2. Use the appropriate issue template
3. Be specific — screenshots and steps to reproduce are always helpful
